# Pong Game

A basic pong game using SVGs.

![Screen Shot Ping Game](public/images/screen-shot-pong.png "pong game")

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a : up
* z : down

**Player 2:**
* ▲ : up
* ▼ : down
